# Python La Paz 

Una comunidad de python a mas de 3600 metros de altura.

## 🌍 ¿Quiénes somos?

Somos una comunidad open-source que desde el 2016 promueve el uso del lenguaje de programación Python en la ciudad de La Paz y en Bolivia con actividades gratuitas y sin fines de lucro. Nuestras actividades se centran en tres ejes: **investigación**, **educación** y **contribución**.
